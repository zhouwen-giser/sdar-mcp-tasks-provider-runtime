# Home Assistant Light Provider

`@sdar/home-assistant-light-provider` is a standalone Runtime Adapter for explicitly configured
Home Assistant `light.*` entities. See the [Provider guide](../../docs/providers/home-assistant-light-provider.md)
for configuration, deployment, security, and operation examples.

Run locally with `pnpm dev:ha-light` after configuring the environment and a token file. The
Provider never accepts a Home Assistant token directly from an environment variable.
